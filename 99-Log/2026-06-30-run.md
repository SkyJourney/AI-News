### Phase 4 writer 1 个非阻塞错误

```
seen_urls_missing_url: http://weixin.sogou.com/weixin?type=2&amp;query=...（jiqizhixin "AI 助手主动交互" 那条）
```

- **影响**：Zettel `202606300952-ai-assistant-proactive-interaction.md` 已建好，但 zettel_id 没回填到 _seen-urls。下次跑 filter 仍会按 first_seen 时间窗判定（不会少做去重），无功能性损失。
- **修复建议**：news-cluster agent system prompt 应明确"URL 字段保持原始字符不转义"，或 cluster-merge.py 在写 cluster.json 前做 `&amp;` → `&` 反转换。
