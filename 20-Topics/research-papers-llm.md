# LLM 研究论文 Research Papers - LLM

追踪专注于大语言模型训练、解码、评测、Agent 能力等方向的研究论文。

## 2026-06-27

- **Reinforcement Learning without Ground-Truth Solutions can Improve LLMs** ([[202606271809-river-no-ground-truth-rl]])
  RiVER 框架首次系统验证无 ground-truth RL 可提升 LLM；Qwen3-8B 与 GLM-Z1-9B 在 ALE-Bench +9%。
  源：`arxiv-api`

- **When are likely answers right? Sequence Probability and Correctness in LLMs** ([[202606271810-seq-probability-correctness-llm]])
  跨解码方法量化验证：高概率序列不等于正确答案，动摇基于概率过滤的主流做法。
  源：`arxiv-api`

- **Empowering GUI Agents via PEEU** ([[202606271811-peeu-gui-agent-7b]])
  PEEU 框架让 7B GUI Agent 以 30.6% 优势超越 Qwen2.5-VL-32B，高层次任务训练增强 OOD 泛化。
  源：`arxiv-api`

- **When Does Combining Language Models Help? Co-Failure Ceiling Across 67 Frontier Models** ([[202606271812-multi-model-co-failure-ceiling]])
  67 模型规模验证多模型组合收益上限由共同出错率决定；实测低估 2.5x。
  源：`arxiv-api`

- **Blackwell Approachability and Gradient Equilibrium are Equivalent**
  GEQ 与 Blackwell approachability 算法等价，理论博弈学习成果。
  源：`arxiv-api`

- **Designing Reward Signals for Portable Query Generation**
  RLAIF 框架；rule-based reward floor 缓解 verbatim copying 问题。
  源：`arxiv-api`
