# F2.4 · Lumina → Quartz 5 tokens 映射表

日期：2026-07-01
上游：`web/poc-quartz/quartz.config.yaml`（POC 期暖橙实验）+ [[quartz-customization-assessment]] 5 层入口分析 + `scratchpad/lumina-refs/lumina-tokens-authoritative.md`（Lumina tokens 权威）
用途：F2.4 P2「视觉层分层覆盖」的唯一改动清单

## TL;DR

Lumina 有 **45+ 色板 tokens** 和 **6 阶 typography**，Quartz 标准 config 只提供 **9 项色板 + 3 类字体**。策略：

- **9 项 Quartz 标准变量**（`--light` / `--dark` / `--secondary` 等）→ 用 `quartz.config.yaml` 的 `theme.colors.lightMode` **直接覆盖为 Lumina 值**（推翻 POC 暖橙）
- **Lumina 额外的 36 项色板 + 玻璃拟态 + 圆角阶梯 + AI 语义**（`--surface-container-high` / `--ai-accent` / `--glass-blur` / `--radius-2xl` 等）→ **新增 CSS 变量**，写在 `quartz/styles/custom.scss` `:root {}` 里
- **utility class**（`.glass-panel` / `.ai-accent-bar` / `.reading-well` / `.masonry-grid`）→ 写在 `custom.scss`
- 组件层（Header / Backlinks / Explorer）→ P3 `quartz.ts` TS override 换自定义 preact 组件

**关键 upgrade-safety**：所有改动只落在 `quartz.config.yaml` + `quartz/styles/custom.scss` + `quartz.ts` + 独立组件文件，**不动 base.scss / variables.scss / core**——`npx quartz upgrade` 无冲突。

## Part 1 · quartz.config.yaml 改动（Quartz 标准 9 项）

### 色板（lightMode）— 推翻 POC 暖橙

```yaml
configuration:
  theme:
    colors:
      lightMode:
        # Lumina 语义映射
        light: "#faf8ff"         # Lumina surface（冷调 near-white）
        lightgray: "#c3c5d9"     # Lumina outline-variant（弱分隔线）
        gray: "#737688"          # Lumina outline（元数据文字）
        darkgray: "#434656"      # Lumina on-surface-variant（次文本）
        dark: "#131b2e"          # Lumina on-surface（Deep Navy 主文本）
        secondary: "#003ec7"     # Lumina primary（Electric Blue，主链接）
        tertiary: "#0052ff"      # Lumina primary-container（次链接 / hover）
        highlight: rgba(0, 82, 255, 0.10)   # Lumina primary 10% 底
        textHighlight: "#6cf8bb88"          # Lumina secondary-container（Emerald）50%
      darkMode:
        # 暗色模式先用反相占位，F2.4 P2 结尾补正式暗色板
        light: "#131b2e"
        lightgray: "#283044"
        gray: "#737688"
        darkgray: "#c3c5d9"
        dark: "#faf8ff"
        secondary: "#b7c4ff"     # Lumina inverse-primary
        tertiary: "#dfe3ff"      # Lumina on-primary-container
        highlight: rgba(183, 196, 255, 0.15)
        textHighlight: "#4edea388"
```

### 字体

```yaml
configuration:
  theme:
    fontOrigin: googleFonts   # P2 决策点：改 local 需挂载 .woff2 到 public/fonts/
    cdnCaching: true
    typography:
      header: Geist            # 之前 Schibsted Grotesk
      body: Inter              # 之前 Source Sans Pro
      code: IBM Plex Mono      # 保留
```

## Part 3 · Quartz 组件 → Lumina 组件 映射

P3 阶段 `quartz.ts` TS override 用到。这里先记规划，P3 时对号入座实现：

| Quartz 现有组件 | Lumina 对应 | 改动方式 | Phase |
|---|---|---|---|
| `Component.Header` | Top NavBar（fixed 80px 玻璃）| override 用自定义 `LuminaHeader` preact | P3 |
| `Component.Search` | Glass Search Bar (Cmd+K) | 保留 Quartz Search，套 `.lumina-panel` 样式 | P3 |
| `Component.Explorer` | 左侧栏（Sources 屏 Left Margin Meta 参考）| 深度改造 or 替换为 `LuminaSidebar` | P3 |
| `Component.Backlinks` | Related Intelligence 分栏（按目录分组）| 完全新写 `LuminaGroupedBacklinks` | **P4** |
| `Component.PageTitle` | Hero title `display-lg` + subtitle | 简单 override | P3 |
| `Component.TagList` | Contextual Chips（`.lumina-chip`）| CSS-only 覆盖 | P2 |
| `Component.ContentMeta` | label-caps 元数据行 | CSS-only 覆盖 | P2 |
| `Component.Footer` | Lumina Footer（`bg-surface-container-low`）| CSS-only 覆盖 | P2 |
| Quartz Popover（enablePopovers）| Popover Preview（玻璃拟态卡）| CSS-only 覆盖 + 内容渲染增强 | **P4** |
| `Component.Graph` | Vault Graph CTA（Topics 屏）| 保留 Quartz Graph，套 Lumina 边框 | P3 |
| `Component.TableOfContents` | 右侧栏 Article Progress 组件 | override 增强 | P3 |

新增 Lumina-only 组件（无 Quartz 对应，纯新写）：
- **FloatingDock**：底部主导航（**依赖用户 Nav 决策**）
- **AISummaryToggle**：卡片内 AI 摘要 unfurl 组件
- **IntelligenceHighlight**：文中重点高亮（Sources 屏）
- **SourceIntegrityBadge**：来源可信度徽章（Sources 屏）
- **TranslationToggle**：中英切换（Sources 屏）
- **ZettelIDChips**：`#Z-8821` 短 tag 双链索引（可作为 Related Intelligence 补充）
