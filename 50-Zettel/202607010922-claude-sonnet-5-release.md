## 概念 / 事件

Anthropic 发布 Claude Sonnet 5：在编码、Agent 工作流与专业任务上达到前沿性能（frontier performance），强调规模化部署能力。

## 关键洞察

- "前沿性能 at scale"是关键定位：Sonnet 5 不是参数最大的模型，而是在可用性与能力之间取得最优平衡的产品级模型
- 编码 + Agent 双重强调说明 Anthropic 将 agentic coding 场景视为 Sonnet 系列的核心差异化战场（对比 Opus 的研究/推理定位）
- 与 Claude Science（[[202607010913-claude-science-workbench]]）同日发布，Anthropic 在模型层与应用层同步推进，加快垂直渗透
