## 概念 / 事件

PEEU 框架让 7B GUI Agent 以 30.6% 优势超越 Qwen2.5-VL-32B，高层次任务训练显著增强 OOD 泛化。

## 关键洞察

- 7B 超越 32B 是效率突破信号，PEEU 的策略（高层次任务训练）可能对其他 agent 场景同样有效
- OOD 泛化提升说明 PEEU 训练出了更抽象的任务理解，而非记忆特定界面
- GUI Agent 是落地场景极强的方向，30.6% 提升在实用价值上非常显著
