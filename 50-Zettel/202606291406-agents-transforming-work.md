## 概念 / 事件

OpenAI 发布 AI agent 生产力研究论文，数据显示 agent 正在扩展人类工作边界——支持更长、更复杂的任务流，并跨角色提升生产力。

## 关键洞察

- 研究首次以量化方式呈现 agent 对任务完成时长与复杂度边界的影响，而非仅描述"有用"
- "更长任务"是当前 agent 价值的核心维度，与传统 co-pilot（辅助单步）形成范式区分
- 跨角色生产力提升意味着 agent 的赋能效应已超出软件工程师单一职能，向知识工作者泛化
