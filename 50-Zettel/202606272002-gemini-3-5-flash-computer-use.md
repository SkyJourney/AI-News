## 概念 / 事件
Google DeepMind 将 computer use 能力下放到效率级 Gemini 3.5 Flash 模型，实现 agent 界面操作能力的民主化。

## 关键洞察
- computer use 从旗舰级扩展到效率级模型，显著降低使用成本门槛
- Flash 系列定位轻量高效，此举意味着 agent 自动化 GUI 操作不再依赖昂贵旗舰模型
- 与 GPT-5.6 Sol 同日发布，Google/OpenAI 进入密集竞争节奏
