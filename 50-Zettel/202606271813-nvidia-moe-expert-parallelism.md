## 概念 / 事件

NVIDIA 官方开源 MoE 专家并行库（Enhanced Transformers），单行 import 即可获得 3.7x 微调加速。

## 关键洞察

- 单行 import 极低的使用门槛意味着 MoE 训练加速将快速普及，不再是大厂专属
- 3.7x 加速来自专家并行（expert parallelism），专注 MoE 架构特性而非通用优化
- NVIDIA 官方出品保证了与 CUDA 生态的深度集成，可信度与长期维护质量较高
