## 概念 / 事件
OpenAI 与 Broadcom 联合发布 Jalapeño，首款专为 LLM 推理优化的定制 ASIC，标志 OpenAI 向硅层垂直整合迈出关键一步。

## 关键洞察
- Jalapeño 是 OpenAI 首款自研推理芯片，战略意义在于降低对 NVIDIA GPU 的长期依赖
- 选择与 Broadcom 合作而非独立流片，延续大公司定制 ASIC + EDA 合作的主流路线
- LLM 推理专用架构意味着针对 Transformer decode 阶段的内存带宽与功耗优化
