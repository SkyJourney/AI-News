## 概念 / 事件

Interconnects Open Artifacts #22 纵览 2025 年下半年开源生态：NVIDIA Nemotron-3 Ultra、Cohere Command A+、GLM-5.2、Zyphra ZAYA1、Poolside Laguna-M.1 同期发布，来自纯模型公司、大厂与产品公司的三类动机同台竞争。

## 关键洞察

- 开源模型发布主体已多元化为三类：纯模型公司（靠权重换影响力）、大厂（靠开源建护城河）、产品公司（靠开源降获客成本）——动机差异决定模型定位
- ZAYA1（Zyphra）与 Laguna-M.1（Poolside）代表新入局者；Nemotron-3 Ultra 和 Command A+ 代表大厂防御性开源
- GLM-5.2 连续出现在两周榜单，开放 Agent 性能领先信号持续强化（参见 [[202606271432-glm-52-open-agents]]）
