## 概念 / 事件

OpenAI 发布 Daybreak 安全套件，含专用网络安全模型 GPT-5.5-Cyber 与 Codex Security 工具，首次将 AI 能力系统性导向漏洞发现、验证与修复。

## 关键洞察

- GPT-5.5-Cyber 是 OpenAI 首款定向网络安全场景的专用模型，不是通用模型的附属功能
- Codex Security 将漏洞扫描流程自动化，覆盖"发现 → 验证 → 修复"全链路
- 同日发布 Patch the Planet 子项目，为开源维护者提供 AI 辅助漏洞修复，将安全能力向开源生态开放
- 与 GPT-5.6 Sol 同期的"最先进安全栈"绑定发布，暗示 OpenAI 将 AI 网安能力作为下一代模型的配套基础设施
