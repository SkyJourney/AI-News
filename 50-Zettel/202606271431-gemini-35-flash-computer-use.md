## 概念 / 事件

Google DeepMind 在 Gemini 3.5 Flash 中引入 computer use 能力，使轻量模型可操控桌面与浏览器。

## 关键洞察

- Gemini 正式进入 computer use 赛道，直接对标 Claude 的 computer use API
- Flash 系列的"轻量 + 快速"定位与 computer use 组合，暗示 Google 押注端侧自动化而非纯云推理
- 发布时间（2026-06-24）紧随 OpenAI GPT-5.6 Sol，模型发布节奏明显加速
