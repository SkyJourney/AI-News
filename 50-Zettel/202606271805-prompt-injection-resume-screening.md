## 概念 / 事件

招聘领域首次系统性研究 prompt injection：个体攻击有效，但群体大规模部署时系统发生自我崩溃。

## 关键洞察

- 招聘场景高度高风险——简历筛选 LLM 面对真实攻击者动机，是 prompt injection 最危险的落地域之一
- 群体崩溃现象（系统大规模注入后自我失效）是新发现，意味着攻击存在内生制约，也意味着防御有新抓手
- 首次系统研究确立了该领域的安全基线，后续工作可直接对比
