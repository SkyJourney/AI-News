## 概念 / 事件
招聘 LLM 系统中 prompt injection 呈现"个体有效 → 群体崩溃"的双重效应，是可复用的 AI 安全失败案例。

## 关键洞察
- 少量注入对个体简历有效（攻击者受益），但广泛注入导致整体排名混乱（群体崩溃）
- 说明 LLM 招聘系统在系统层面对 prompt injection 缺乏健壮性，不仅是模型问题
- 兼具攻击实用案例与政策含义，可直接用作 AI 安全教学与监管参考材料
