## 概念 / 事件

智谱 AI 发布 GLM-5.2，Interconnects 评价其为开源 agent 赛道的阶跃式突破（step change）。

## 关键洞察

- GLM-5.2 在多个 agent benchmark 上首次让开源模型与闭源前沿模型的差距显著缩小
- "step change"定性来自 Interconnects（Nathan Lambert），具有较高评估可信度
- 开源 agent 模型能力提升对下游工具链生态（如 LangChain、AutoGen）有直接促进效应
