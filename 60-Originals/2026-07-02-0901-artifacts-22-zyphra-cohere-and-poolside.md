# 最新开源模型成果（#22）：Zyphra、Cohere 与 Poolside 拓展生态版图

## 概览

开源模型的生态正在变得越来越多样化，全球越来越多的组织发布各种不同的模型。这代表了一个重要转变——相比前一年中国公司主导市场的情况，现在的局面已大不相同。

## 模型发布背后的动机

文章确定了发布模型的组织的三个主要类别：

**"纯粹"的模型制造者**

这些组织专注于前沿级别的模型训练，包括中国公司如 DeepSeek（深思），以及西方公司如 Poolside 和 Zyphra（第菲拉），以及主权 AI 参与者如 Cohere（科赫雷）和 Mistral（翠鸟）。

**大科技公司**

如阿里巴巴（Qwen 千问）和谷歌（Gemma 双子）等公司战略性地使用开源发布——阿里巴巴推广其高级闭源模型，而英伟达则受益于 GPU 需求的增加。

**产品公司**

JetBrains 和 Zed 等组织发布专门的小型模型，专门针对其产品需求定制，不会损害其业务模式。

## 精选模型

**英伟达 Nemotron-3-Ultra-550B** 采用 LatentMoE（潜在混合专家）技术以提高速度，并采用专门为模型权重设计的 OpenMDW（开放模型权重）许可证。

**Cohere 的 Command A+**（218B-A25B 混合专家）现已采用 Apache 2.0 许可证，从之前的非商业限制转向开放。具有多模态、多语言和智能代理能力。

**GLM-5.2** 展示了真实的日常任务可用性，性能与领先的闭源模型相当。

**Zyphra 的 ZAYA1** 提供 74B 和 8B 两种模型变体，基于 AMD GPU 构建。

**Poolside 的 Laguna-M.1** 承诺在 Apache 2.0 许可证下持续发布开源权重。

## 关键洞察

生态的多样性加强了开源开发，并表明限制这一领域的发展只会将 AI 进步集中在少数几个参与者手中。
